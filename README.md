# IntroCSharpBackend

The goal of this repo is to provide a solid foundation in the .NET ecosystem, focusing on C# as the primary programming language. We'll cover the core concepts, best practices, and basic architecture patterns commonly used in real-world applications. This initiative aims to help new developers ramp up quickly, write clean and maintainable code, and contribute more effectively to our projects.

The code shared throughout the sessions includes detailed comments to give students clear context on what’s being done and why. Additionally, the repository contains markdown files with extra explanations and references, so they can read more about the concepts discussed in class at their own pace.

Some of the topics of this foundation class:

- How to declare classes
- Constructors
- Static keywords
- Pure vs Impure Functions
- Variable types
- Object Oriented Programing
- Operators and Expresions
- Collections
- Linq
- High order functions and Delegates
- Lambda Expressions
